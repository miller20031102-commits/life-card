# 人生副本 v15.7｜完整報告吸引力優化版

本版保留 v15.6 的無限抽卡、v15.5 的角色配對修正，以及既有綠界付款與完整報告內容。

## 這次改進

- 結果頁的付費框改成角色專屬預覽，不再只是通用功能清單。
- 會顯示抽到的角色名稱、稀有度與一小段專屬解析。
- 清楚呈現 10 個章節、7 日任務、30 天養成與同角色不重複付費。
- 加入戀愛攻略、隱藏 Boss、升級任務三個內容預覽。
- CTA 改成「解鎖這張角色的完整報告」，更貼近使用者當下結果。
- 明確標示一次解鎖、不是訂閱制、綠界安全付款。
- 付款確認視窗增加角色專屬摘要與價值說明。
- 中段付費介紹改成 6 個具體內容面向。
- 移除「未來可做實體卡」這種不直接增加當下價值的文字。
- 不使用倒數、假優惠、假名額或強迫式購買設計。

只需要覆蓋 `website` 資料夾內的 8 個網站檔案。
Cloudflare Worker、D1 與綠界設定不需要修改。
