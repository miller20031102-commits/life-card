# 人生副本｜角色卡販賣機

這是一個純前端靜態網站，可以直接上傳 GitHub Pages。

## 功能

- 12 題角色測驗
- 20 種角色卡結果
- 稀有度、技能、弱點、戀愛屬性、社交定位、工作模式
- 下載 PNG 角色卡
- 複製 Threads 分享文
- 本機收藏角色卡
- 付費完整報告預告區
- 手機版 RWD
- SEO / Open Graph 基本設定

## 檔案

- `index.html`：網站結構
- `style.css`：完整視覺設計
- `script.js`：測驗邏輯、角色資料、下載圖片功能

## 上傳 GitHub Pages

1. 建立一個新的 GitHub repository
2. 把這三個檔案上傳到 repository 根目錄
3. 到 Settings → Pages
4. Source 選 `Deploy from a branch`
5. Branch 選 `main` / root
6. 等 1～2 分鐘後開啟 GitHub Pages 網址
