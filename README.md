# 人生副本 v16.0｜首頁自然化版

本版調整付款資訊出現的位置，避免首頁第一眼過度商業化。

## 已修改

- 移除首頁主視覺旁／下方的綠界、Visa、Mastercard、JCB、Apple Pay 圖示。
- 首頁不再直接顯示 NT$49，改成「33 種角色結果」。
- 首頁亮點改為「免費測驗不綁卡」，先讓使用者放心測驗。
- 付款品牌圖示只保留在：
  1. 抽卡結果的完整報告解鎖框
  2. 最後付款確認視窗
- 中段價格框改成一行低調的「綠界安全付款」說明，不再重複排列四個品牌圖示。
- 結果頁的付款圖示縮小並降低存在感，避免搶過完整報告內容。
- 保留 v15.9 圖示檔、v15.7 付費框、v15.6 無限抽卡與 v15.5 配對修正。
- 快取版本更新為 v16.0。

部署時請將 `website` 資料夾內所有檔案與 `payment-icons` 資料夾一起上傳。
Cloudflare Worker、D1、綠界設定不需要修改。
