# 人生副本｜正式版

這是可上傳到 GitHub Pages 的正式版。

上傳檔案：
- index.html
- style.css
- script.js
- og-image.png
- README.md

功能：
- 手機版測驗流程
- 角色卡生成
- JPG 圖片預覽與長按儲存
- 我的卡冊
- 付費完整報告
- 每張卡可獨立解鎖
- 已解鎖報告可在同裝置同瀏覽器保存查看


## v3 更新
- 新增每日免費抽卡 2 次限制。
- 我的卡冊與已解鎖報告不消耗抽卡次數。
- 次數以同一台裝置 / 同一個瀏覽器的本機紀錄為準，隔天自動恢復。


## v5 更新
- 完整報告區新增「選擇要解鎖的卡」。
- 使用者可從我的卡冊挑選任一張卡解鎖，避免誤解鎖目前頁面上的卡。
- 單張卡 NT$49 建議先作為早鳥價測試。
