# 人生副本 v16.4｜雙人卡冊永久解鎖版

## 修正

- 雙人免費試算完成後會自動存進「我的卡冊」。
- 卡冊新增「單人角色卡」與「雙人副本」兩個分頁。
- 同一組角色不會因交換 A／B 順序而重複保存。
- 同一組角色可在卡冊保存朋友、情侶、曖昧、搭檔四種模式。
- 卡冊會顯示最後查看模式、隊伍名稱、免費默契區間或已解鎖精確分數。
- 從卡冊可以直接恢復雙人配對、模式與完整內容。

## 永久解鎖

- 完整雙人副本 NT$10，權限仍由 Cloudflare D1 後端保存，沒有到期時間。
- 同一組角色付款一次，四種關係模式全部永久解鎖。
- 付款後四種模式會一起加入雙人卡冊。
- 清除瀏覽器或換裝置後，使用原本的私人備份碼同步，雙人副本會重新建立並恢復永久解鎖。
- 免費、未付款的雙人試算只存在目前瀏覽器；清除資料後無法從後端恢復。

## 部署

本版 Worker 與 v16.3 相同，不必重新部署。
只需要覆蓋 `website` 資料夾內的網站檔案。
