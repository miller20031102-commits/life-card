# 人生副本 v15.2｜每日次數修正版

本版修正朋友「今天沒玩但次數顯示 0」的問題。

## 原因

舊版每日紀錄只存 count。最後一題快速點兩下或重複事件同時觸發時，
可能把同一場測驗記錄兩次，直接由 2 次扣成 0 次。

## 修正

- 每次測驗使用唯一場次編號。
- 同一場測驗最多只扣一次。
- 防止選項快速連點、跳題與最後一題重複結算。
- 每日紀錄改為場次清單，不再只存容易錯寫的 count。
- 每日重置固定使用台灣時間 00:00。
- 使用新的 localStorage 紀錄鍵，讓已被舊版錯扣的使用者恢復今日 2 次。
- index.html 加入 v15.2 快取版本，避免瀏覽器繼續使用舊 script.js。

Worker、D1、綠界 Secrets 與付款設定不需要變更。
