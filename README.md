# 人生副本｜正式版 v9

這版已設定付款回報表單，使用者付款流程為：

1. 使用者選擇要解鎖的角色卡。
2. 點「前往付款回報表單」。
3. 在表單內查看銀行轉帳資訊並填寫結果編號、付款後五碼、聯絡方式。
4. 你確認款項後，從解鎖碼 Google Sheet 發一組 UNUSED / SOLD 解鎖碼給使用者。
5. 使用者回網站輸入解鎖碼，成功後該張卡會永久保存在卡冊。
