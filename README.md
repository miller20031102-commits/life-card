# 人生副本 v15.5｜答題與角色卡配對修正版

本版針對 12 題答案到 33 張角色卡的配對邏輯做完整稽核。

## 原版發現

- 原本使用「原始分數 × 角色向量」直接相乘，角色向量數值較大的卡會天然佔優勢。
- 稀有度額外加分，導致 UR／SSR 並非因為更符合答案，而是因稀有度被加分。
- 16,777,216 種完整答題組合中，有 7 張卡完全不可能抽到。
- 人生重開勇者單張約佔 35%，部分卡只有十萬分之一以下。

## v15.5 修正

- 先將每個人格軸依題目分布標準化，避免某些軸因題目較多而天然權重過高。
- 將角色向量依 33 張卡的分布標準化，再以餘弦相似度找最符合的角色。
- 稀有度不再影響人格配對。
- 加入角色專屬關鍵答案與組合條件，區分相近角色，例如：
  - 戀愛逃跑法師／想愛不敢愛幽靈／戀愛腦封印師
  - 社交電量獵人／社恐派對幽靈／被動社交貓
  - 情緒觀察系巫師／情緒雲朵水母／朋友群補血師
- 只保留極小的固定亂數處理完全同分；相同答案永遠得到相同結果。

## 全組合測試

- 完整枚舉：4^12 = 16,777,216 種答題組合
- 可抽到角色：33 / 33
- 完全抽不到：0
- 單一卡牌最低約：0.83%
- 單一卡牌最高約：6.28%

這是所有答案等機率下的程式測試分布，不代表實際使用者分布。
