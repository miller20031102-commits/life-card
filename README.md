# 人生副本 v16.3｜雙人副本完整除錯版

本版針對雙人副本做完整除錯與差異化。

## 已修正

- 付款解鎖後不再同時顯示「免費分數區間」與「精確分數」，降低重複和卡片高度。
- 雙人卡的 `color-mix()` 改為 html2canvas 可處理的背景，避免 JPG 生成失敗。
- JPG 改用固定 760px 匯出版型，不會因手機與電腦螢幕不同而排版亂掉。
- 依裝置限制自動調整畫布倍率，避免 iPhone 因圖片過長超過 Canvas 上限。
- 儲存視窗新增「下載 JPG」與「分享圖片／儲存到相簿」。
- 修正長標題、窄螢幕、七日任務與奇數內容卡的排版。
- 關閉圖片視窗後會釋放 Blob URL，避免多次生成造成記憶體累積。

## 雙人內容差異化

每一組角色與每一種關係模式，都會引用雙方角色的名稱、技能、被動、弱點、任務與模式權重。以下欄位皆逐組生成：

- 隊伍名稱
- 免費摘要
- 合體技能
- 最強優勢
- 最容易卡關
- 修復攻略
- 最適合情境
- 分數判定原因
- 本週雙人任務
- 7 日任務
- 隱藏設定

完整測試範圍：33 張角色 × 561 組不重複配對 × 4 種模式，共 2,244 組結果。

## 部署

本版 Worker 金額與權限邏輯未變，仍為：

- 單人完整報告 NT$49
- 完整雙人副本 NT$10

只需更新 `website` 資料夾。Worker 可保留 v2.3；完整包仍附上相同 Worker 供備份。
