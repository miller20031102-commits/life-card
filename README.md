# 人生副本｜表單付款正式版

這版付款流程改成：網站不顯示銀行帳號，使用者會前往 Google 表單查看銀行帳號與填寫付款回報。

## 上線前必改

打開 `script.js`，找到：

```js
paymentFormUrl: 'https://forms.gle/ck8NkqScfuUNbysn8'
```

改成你的付款回報 Google 表單連結。

## Google 表單建議欄位

1. 暱稱
2. 聯絡方式（Email / IG / Threads）
3. 要解鎖的角色名稱
4. 結果編號
5. 結果編號
6. 付款金額
7. 付款後五碼
8. 轉帳截圖上傳
9. 同意事項確認

## 表單最上方建議提醒文字

請先確認你要解鎖的角色卡與結果編號，再依下方銀行帳號轉帳 NT$49。轉帳完成後，請填寫付款後五碼並上傳截圖。客服確認後會提供一組專屬解鎖碼。

銀行：你的銀行名稱  
銀行代碼：你的銀行代碼  
帳號：你的帳號  
戶名：你的戶名  
金額：NT$49

## 表單同意事項建議

本服務為娛樂性角色卡生成與個人化文字內容，不構成心理諮商、醫療診斷、法律、投資或人生決策建議。完整角色報告屬於客製化數位內容／線上服務。解鎖碼一經使用並顯示完整報告，即視為服務完成。除系統錯誤、重複付款或未收到解鎖碼外，恕不接受退款。
